# Plan - Filter Exporter Chapters (en-ko)

동일한 챕터에 대해 `.en-ko.md` 파일과 일반 `.md` 파일이 모두 존재할 때, `.en-ko.md` 파일만 선택하여 내보내는 기능 구현 계획입니다.

## 1. 목적 및 필요성
- 뷰어 내보내기(Exporter) 과정에서 한글 번역본(`.en-ko.md`)이 준비된 경우 원본(`.md`)은 무시하고 번역본만 내보내어 중복 내보내기를 방지하고 최종 번역본만 사용자에게 제공되도록 합니다.

## 2. 작업 상세 계획
- `apps/viewer/src/exporter/utils/fileLoader.ts`의 `loadBookFromDirectory` 함수를 수정합니다.
- 디렉터리에서 로드된 `.md` 파일들 중에서 `.en-ko.md` 접미사가 존재하는 파일들의 베이스 이름을 추출합니다.
- 일반 `.md` 파일 중에서 동일한 베이스 이름을 가진 번역본 `.en-ko.md` 파일이 존재할 경우 해당 일반 `.md` 파일은 로딩 대상에서 제외시킵니다.

## 3. 검증 계획
- 변경 전과 후의 동작을 유닛 테스트 또는 로컬 실행(모의 테스트 등)을 통해 확인합니다.
