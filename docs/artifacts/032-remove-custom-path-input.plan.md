# Plan: 서적 경로 직접 지정 입력창 제거 (Remove Custom Path Input)

## 배경 및 목적
현재 서적 내보내기 화면의 설정 폼 카드에는 드롭다운 스캔 목록 외에 사용자가 직접 절대 경로를 입력할 수 있는 입력란이 존재합니다. 그러나 사용자 사용성 단순화 및 드롭다운 선택 방식의 정합성을 위해 해당 기능을 제거하여 UI를 직관적이고 미니멀하게 개선합니다.

## 변경 계획

| 파일 경로 | 액션 | 상세 설명 |
| :--- | :--- | :--- |
| `apps/viewer/src/frontend/src/views/ExporterView.vue` | 수정 | 1. 템플릿(HTML) 내의 `"또는 아래에 전체 경로를 직접 지정할 수 있습니다:"` 텍스트 및 `<input type="text" v-model="customPath" ...>` 태그 삭제.<br>2. 스크립트 단의 `customPath` 상태 변수 삭제 및 내보내기 핸들러(`startExport`)에서 서적 경로 판별 로직을 `selectedBook.value`로만 단순화. |

## 기대 효과
- 화면에 불필요한 정보 입력을 유도하는 창을 없앰으로써 UI 레이아웃의 군더더기를 제거하고, 사용자 조작 실수를 원천 방지함.
