# 098-update-translate-pipeline.review.md

본 리뷰 문서는 번역 파이프라인 프롬프트 지침의 변경 전후를 대비하여 검토한 결과입니다.

## 🔍 변경 사항 대비 및 검토

### 1. 복습용 Q&A 제거
* **기존**: 옵션 A 번역 진행 시 하단에 영-한 병기 복습용 Q&A 10문항을 번역 결과물 최하단에 생성하는 지침이 존재하였습니다.
* **변경**: 번역 결과물 내 Q&A 생성 지침을 완전히 제거하였습니다.
* **이유**: 단순 번역 목적에서 불필요한 Q&A 생성을 지양하여 LLM 추론 시간 및 출력 토큰 낭비를 줄이고 번역 가독성을 높입니다.

### 2. `*.summary.md` 요약 문서 생성
* **기존**: 원본 파일명에 대응하는 파일로 `[원본파일명].en-ko.md` 또는 `[원본파일명].ko.md`만 지침에 기술되어 있었습니다.
* **변경**: 옵션 A를 선택할 경우, 본문 번역이 끝난 뒤 `[원본파일명].summary.md` 파일을 생성하여 각 단락에 대해 `영어 원문 요약 (Original Summary)`과 `한글 번역 요약 (Translated Summary)`이 1:1 매칭되는 요약본을 별도 생성하도록 작성 규칙을 명문화하였습니다.
* **이유**: 사용자가 복습 혹은 정보 속독을 할 때 각 단락의 요약 대비군을 참조하여 더 효율적으로 기술 문서를 읽을 수 있도록 지원하기 위함입니다.
